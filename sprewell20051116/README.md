##### Goto 00_NodeJS_testEnv
```
$ cd [Your Project Path]/00_NodeJS_testEnv
```

##### Install webpack
```sh
$ npm install --save-dev webpack
```
#### Install bable
```sh
$ npm install babel-core babel-loader webpack-dev-server babel-preset-es2015 babel-polyfill --save-dev
```
#### Build Modules
```js
$ npm run build
```
#### Start Server
```js
$ npm start
```
### Open localhost
```
http://localhost:3000/
```
